This project presents an AI-based advanced teaching application that enables live interactive sessions with real-time multilingual support. Teachers can conduct live classes while students join instantly and access features like AI-powered subtitles, voice translation, and smart note generation in PDF/PPT formats. The system uses AI to convert lectures into multiple languages, generate live notes, and provide an interactive Q&A environment. It also includes recording and sharing functionality, making learning more accessible, personalized, and efficient.
